# The Price of Free: How Much Is Your Data Worth?

An interactive educational platform exploring the intersection of Artificial Intelligence, data privacy, fairness, and digital ethics.

 **Live Demo**: [https://the-price-of-free.vercel.app](https://the-price-of-free.vercel.app)

## About This Project

"The Price of Free" is a comprehensive web application designed to educate users about:

- **AI & Machine Learning** - How AI systems work and are trained using data
- **Data Privacy** - Understanding what data is collected and how it's used
- **Fairness & Bias** - Recognizing algorithmic bias and its real-world impact
- **Digital Footprint** - Managing your online presence and personal information
- **Interactive Learning** - Engaging game and reflection exercises

This project was created as an educational initiative to raise awareness about AI ethics and help users make informed digital choices.

## Features

 **Interactive Game** - "Would You Trade It?" experience  
 **Educational Content** - Deep dive into AI, data, and privacy  
 **Beautiful UI** - Modern, responsive design with smooth animations  
 **Pledge System** - Users can make and share digital privacy commitments  
 **Team Gallery** - Meet the creators behind the project  
 **Mobile Responsive** - Fully optimized for all devices  

## Tech Stack

- **Frontend**: [Nuxt 3](https://nuxt.com/) + Vue 3
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL with Prisma ORM
- **Backend**: Nuxt Server Routes
- **Hosting**: [Vercel](https://vercel.com/)
- **UI Components**: Nuxt UI

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- PostgreSQL database

### Installation

1. Clone the repository:
`bash
git clone https://github.com/ijatt/sulam-group-2.git
cd sulam-group-2
`

2. Install dependencies:
`bash
npm install
`

3. Set up environment variables:
`bash
cp .env.example .env
`

4. Generate Prisma client:
`bash
npx prisma generate
`

## Development

Start the development server:
`bash
npm run dev
`

## Building for Production

`bash
npm run build
`

## Deployment
